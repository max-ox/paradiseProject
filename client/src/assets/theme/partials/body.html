Welcome to Orbit

A Scifi Bootstrap 4 HTML Theme

You have been looking for a truly modern Scifi theme for your browser game or unique website design? Look no futher!

This Scifi theme based on Bootstrap 4 is not simply a reskin with some new colors. It changes the complete look and feel of Bootstrap with transparencies, gradients, glow effects and more. And all of that with pure CSS! No strange image assets just for a fancy border. So you can use all the Bootstrap components you know and love, but they look really futuristic.

The only tradeoff is that it is NOT compatible with ancient browsers like IE8-9. But I guess you and your customers won't need that anyway, considering you are looking for a futuristic theme. ;)

This is a HTML/CSS theme. This means you can implement it into your project of choice and it is basically compatible with everything. It can be converted to Angular, React, Vue or even into a Wordpress theme.

The Orbit Bootstrap Theme is based on the Orbit SciFi UI Kit by Nice To Must Have. If you need the PSD files, go get it!
Permission to use the design for a commercial theme was granted by the author.

Features

Everything Bootstrap 4 has to offer an more

  • All 30+ Bootstrap 4 elements available
  • 25+ example layouts (including shop pages)
  • Specialized elements, e.g. map controls, mega menu, compact forms, header & card variants
  • Bootstrap utilities (color, grid, positioning, etc.)
  • Fully responsive (including beautiful mobile tables & image utilities)
  • Scifi design based on pure CSS
  • NO unnecessary image assets
  • Includes Font Awesome Free icon font
  • Easily customizable with modern tools (SCSS/SASS, npm, webpack)
  • Mixins for easily expanding complex styles
  • Supported in all modern browsers (no IE8-9)
